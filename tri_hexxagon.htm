TRICOLEUR HEXXAGON  

Copyright (c) 2005 Bill Taylor

The game is played on a 9x9 hexagonal board, with a hole at the center, using this setup:

The marked stones belong to First player, the others belong to Second. The moves are the same as in Hexxagon, and captures are inspired in the Rock-Paper-Scissors game.

  • MOVE SEQUENCE - The move sequence is given by 134444.... moves per turn
  • MOVE - On this turn, the player either:
    • passes
    • grows: adds one friendly stone adjacent to a friendly piece of the same type
    • jumps: moves a piece already on board to an empty cell two positions away (includes jumping over pieces of either type)
  • CAPTURE - after the movement (grow or jump) all adjacent enemy pieces of the same type are removed
  • FLIP - after the movement (grow or jump) all adjacent enemy pieces of the weak type change color
    • for Red, the weak type is Green
    • for Green, the weak type is Blue
    • for Blue, the weak type is Red
  • GOAL - When both players pass (usually because they can no longer move), the game ends
    • wins the player with more friendly stones

A big group of stones of the same color is very fragile near just one enemy stone of the right color. Flexible armies are required here, which means a lot of care to choose which is the best part to develop on each turn.
