There's an important difference between: a score board for a game like Catchup or Xodd, which merely summarizes (redundantly but often conveniently) the current board state "to save the players having to work out the situation at the start of every turn", and: a score board (or written notes or collected beans or whatever) in a game like Cribbage, where the current "board state" (e.g. seeing what cards are currently in play) does not suffice to know the score, because the score depends on an accumulation of points earned from past events which are no longer visible, and so the score would be impossible for the players to work out at the start of every turn.
