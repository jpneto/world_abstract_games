Rules for YariShogi (Copyright (c) Christiaan Freeling)

YariShogi is a Shogi variant invented by Christiaan Freeling. It is played on a 9x7 board, with the last three ranks as the promotion zone. YariShogi uses the same rules as shogi, except for the pieces which are used and a few differences noted below.

These are the differences between Shogi and YariShogi:

  1. YariShogi is played on a 9x7 board, with the last three ranks being the promotion zone. This is the opening setup: 
    
       7    6    5    4    3    2    1  
    +----+----+----+----+----+----+----+    White (White)
  a |\ R/|\ N/|\ N/|\ K/|\ B/|\ B/|\ R/| a 
    +----+----+----+----+----+----+----+    
  b |    |    |    |    |    |    |    | b 
    +----+----+----+----+----+----+----+
  c |\ P/|\ P/|\ P/|\ P/|\ P/|\ P/|\ P/| c 
    +----+----+----*----*----+----+----+
  d |    |    |    |    |    |    |    | d 
    +----+----+----+----+----+----+----+
  e |    |    |    |    |    |    |    | e 
    +----+----+----+----+----+----+----+
  f |    |    |    |    |    |    |    | f 
    +----+----+----*----*----+----+----+
  g |/ P\|/ P\|/ P\|/ P\|/ P\|/ P\|/ P\| g 
    +----+----+----+----+----+----+----+
  h |    |    |    |    |    |    |    | h 
    +----+----+----+----+----+----+----+    
  i |/ R\|/ B\|/ B\|/ K\|/ N\|/ N\|/ R\| i 
    +----+----+----+----+----+----+----+    Black (Black)
       7    6    5    4    3    2    1
  2. Unlike Shogi, Pawns in YariShogi are allowed to give checkmate when dropped.

YariShogi Pieces

  • An 'x' indicates a square which the piece may move or jump to.
  • A line indicates the piece may move any number of squares in that direction.
  • A number indicates the piece may move in that direction up to that number of squares, but not jumping.
