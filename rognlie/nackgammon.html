Introduction

Welcome to the network Nackgammon server. Nackgammon is played exactly the same as Backgammon except for the opening setup of pieces. Read the Backgammon help page for all the commands and rules.

This is the opening setup for Nackgammon. One piece is taken from each of the two points which have five in Backgammon, and are placed on the point which is the second farthest from the home.
