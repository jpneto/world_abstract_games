Introduction

Welcome to the network Hypergammon server. Hypergammon is played exactly the same as Backgammon except for the opening setup of pieces. Read the Backgammon help page for all the commands and rules.

It consists of only three men - one each on the opponents 1, 2 and 3 points. As such, it makes for a very fast game.
