Introduction

Welcome to the network 8x8Trax server. The commands are the same for all pbmserv games.

8x8Trax is played exactly the same as Trax, except that the board may not grow beyond 8 tiles tall or wide. See the Trax help page for all the rules.

8x8Trax, previously known as stdTrax, was the original version of Trax. Eventually everyone realized that Trax on an infinite board, or "SuperTrax" as it was called, was much better than the original, which was limited to an 8x8 board. So SuperTrax is now the standard Trax, and the original Trax is now known as 8x8Trax.
