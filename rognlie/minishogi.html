Rules for MiniShogi MiniShogi is a Japanese board game based on Shogi. The rules are identical to Shogi except that it is played with fewer pieces on a smaller 5x5 board, and the promotion zone is only the last rank.

The opening setup looks like this: 

      5    4    3    2    1   
    +----+----+----+----+----+    White (player2)
  a |\ R/|\ B/|\ S/|\ G/|\ K/|  a
    +----*----+----+----*----+    
  b |    |    |    |    |\ P/|  b
    +----+----+----+----+----+
  c |    |    |    |    |    |  c
    +----+----+----+----+----+
  d |/ P\|    |    |    |    |  d
    +----*----+----+----*----+    
  e |/ K\|/ G\|/ S\|/ B\|/ R\|  e
    +----+----+----+----+----+    Black (player1)
      5    4    3    2    1

If you do not already know how to play Shogi, then please read the shogi help page to understand the rules of MiniShogi.
