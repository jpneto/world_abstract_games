This game is played on an empty 7x12 square board with the following setup:


MOVE - On each turn, each player moves a stone to an adjacent (orthogonal or diagonal) empty cell. 
All stones of the same color must be connected. If a set of stones is disconnected, all stones except those belonging to the bigger sub-group are removed. 
A move that splits the group into two equal sized sub-groups is forbidden.
For each stone in the 5th row, the player has 2 points. For each stone in the 6th row, 5 points and for each stone on the last row, 8 points.
GOAL - Wins the player that achieves a position worth of 16 points.
