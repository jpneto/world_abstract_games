This game is played on two empty 7x7 square boards.

 
  • DEFINITIONS -
    • Go concepts like group, liberty, Ko are applied here.
    • A swap is to move a stone already on one board, to the other board on the same coordinate, provided that the destination is empty.
  • PIE RULE - One player places three black stones and two white stones on some empty cells. The other player choose sides. White continues.
  • TURN - On each turn, each player must do one of the following things:
    • Passes
    • Swaps a friendly stone and then, optionally, drops a stone. 
      • If no friendly stone can be swapped, the player can swap an enemy stone.
      • It is illegal to create a position with no swaps.
      • Ko is also applied to both boards, it is invalid to make the same swap twice in a row.
  • GOAL - The game ends when both players pass; scoring by Chinese rules.

This game is a mix of Go with a wonderful game, Alice Chess. Notice that the rules permit suicides, which can be useful when creating area for future swaps is more important.
