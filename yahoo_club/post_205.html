It seems to me that there are two slightly different arguments going on here
and that people are talking past each other. I don't believe that anyone is
trying to change the formal definition of "abstract game" as it applies to
the mathematics of combinatorial game theory. Rather, I believe some people
are frustrated with the assertion that the definition used in this
discussion forum must strictly adhere to the formal mathematical definition
and that any game which does not meet all the criteria of this definition
should not be discussed here. In my opinion this is unreasonable. I offer
the following thoughts for for consideration;

1) I'm not an authority but didn't someone post earlier that abstract games
as defined under combinatorial game theory must always end as a result of
one player being unable to move? If this is the case, then games such as
Chess, Twixt, Hex, and many others would, strictly speaking, by this
definition, not be considered abstract games. If an exception is made for
this, then logically it should be acceptable to entertain other exceptions
to the definition as well.

2) I'm assuming that this discussion group is not only for combinatorial
game theorists, although they are of course welcome as far as I'm concerned.
Many people aren't familiar with formal game theory and probably believe
that "abstract game" means a game that is abstract, with the word abstract
itself being defined according to one of its dictionary definitions. This
certainly seems reasonable to me and I have no problem with such a
definition in the context of this discussion group.

3) There are some good themeless (or almost themeless) games with no luck
that support more than two players and I have no problem with such games
being included under the umbrella of this discussion group. It also wouldn't
spoil my day if I saw the occasional post on a game such as Backgammon which
is a themeless game that has some hidden information.

4)This is not a particularly active discussion group these days. If I had to
wade through dozens of posts per day, such as with spielfrieks, then maybe
it would make sense to me to think about narrowing the scope of the group,
but as it is I welcome discussion of any game that can be reasonably called
abstract by either the dictionary definition or the definition as used in
formal game theory.

5) If the disagreements that have been posted here reflect frustration with
seeing posts that violate the strict formal definition of the name of the
group (i.e., "these are interesting but they shouldn't be posted HERE", as
opposed to, "I'm not interested in these and I don't want to have to sort
through them") , then perhaps this could be addressed by making some changes
to the introductory page for the group in terms of defining what is or is
not appropriate material for posting here.

-Mike Schoessow
