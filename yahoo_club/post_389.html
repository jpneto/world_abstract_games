Assuming that you're using WinZip, when you click on the Extract
button a menu will appear. At the lower left of the pop-up there is
an option, "Use folder names". Be sure that this has been selected.

If you using another Un-Zip program, you might read their help
section. If available.

The point being, that the files need to be in their proper folders
for the implementation to work properly. You can do this manually by
creating a folder named "Images", open it and create another folder
named "Zhadu". Then transfer all the BMPs into the Zhadu folder.
