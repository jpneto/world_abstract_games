I been fiddling around with several ideas that I
think is beginning to gel as a game.
The game should
start on a suitably large board. Go-board should be
sufficient.
Though I haven't worked out the details
yet.
First the rules for the turns:
1. Each turn consist
of an optional move and an required placement.

The option to move can only be invoked if one of the
pieces owned
by the player who has the turn is
threatened. Which piece owned by that player
is moved
however is not restricted.
2. Each piece can move
exactly as far orthogonally as there are pieces around it

regardless who those pieces belongs to.
3. A piece (other
than the first piece) can only be placed within or on
a five by five
box centering on an existing
piece. For the first placement, it is arbitrary.
4. A
capture is made when a move takes a piece onto a spot
occupied by its opponent.
captured piece is returned
to the owner of the piece.
Object:
To capture
a set amount of the opponent's stones. Experienced
player can be hadicapped
against weaker players by
changing the number of pieces each side need to win,

say 5 for an experienced player 3 for the less
experienced.
The opening:
Because of the limitation on
placement, the opening might become very well versed
to
experienced players, so after a set number of moves, less
than five is probably
prefered. The two sides get
one chance in the game to switch
sides.
