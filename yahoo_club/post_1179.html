>> Fischer Random Chess is a set of 960 abstract games.
>
> Ok, so then a tournament where there is a set of say 100 abstract
> strategy games and they are picked at random what is played, would be
> akin to Fischer Random Chess?

Yes, you may see it so.
After each selection an abstract game begins.
The random selection is not a part of the selected abstract game.

You can make the selection to an abstract game if you do not choose
randomly but by an decision-making process between both players, e.g.
as Mark Thompson's "Thompson Nonrandom Chess" in a previous post.

>
> In such a tournament, would the fair thing be to have players play
> both sides for each game?

Not necessarily. If each game in the set is balanced (with chances
for both players) then there is no need to play both sides.
In Fischer Random Chess there is no setup known that gives one
player an decisive advantage.
