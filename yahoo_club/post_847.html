> Ice Towers (Abdrew Looney, 2002)

This is a simultaneous-play game. I suppose it could be played as a
turn-based game, but I'm not sure how well it would work. However,
there are plenty of other stacking games using Icehouse pyramids that
are true abstract strategy games:

Volcano (Kristin Looney, 2000)
Stack Chess (Northern Berkshire Game Group, 2001)
Zagami (Kory Heath, 2002)
Sprawl (Jonathan A. Leistiko, 2003)
Moscow Ice (Kendall Johns, 2004)
E, the Game of Martian Chinese Checkers (Carol Townsend, 2004)

These games (and more) can be found either through S.L.I.C.K. or the
Ice Game Design Competition:

http://www.icehousegames.com/SLICK.html
http://www.icehousegames.com/contest/


Outside the world of Icehouse games, Bruno Faidutti invented a simple
stacking game called Pilade in 1993, which was published as Babylone
in 2003. Bruno's page about this game mentions one other stacking
game not on your list: Stapel (CornAi van Moorsel, no date given).

http://www.faidutti.com/index.php?Module=mesjeux&id=323


Greg Lam designed an abstract strategy game named Rock, Scissors, Paper
in 2001. One kind of piece (paper) can be stacked.

http://www.pair-of-dice.com/games/rsp.html
