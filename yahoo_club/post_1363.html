Hi Greg,

I always recommend using a much larger board than the space saving
size I use in the rule sheet. This'll facilitate a richer game and
reduce the first move advantage. I'd recommend whatever size gives
you about as many cells as you have in a Go board as the standard
size. For a faster game I'd recommend 1/2 to 2/3 that number of cells.

I also recommend an even n so that there will be an even number of
cells, mostly for for aesthetic reasons.
