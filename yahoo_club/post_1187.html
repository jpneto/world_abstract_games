Mark, "abstract game" is just a phrase in ordinary English, and has no
technical meaning for the vast majority of the world's population.
The fact there there is so much dispute over the finer points of the
definition just points up the need for a defining document.
Personally, I think a technical field like game theory needs to have
its own nomenclature, but I won't venture what that should be.

A second point: If a game is not a theoretical draw, then there has to
be an advantage for one player or the other (obvious). There are
several potential problems here: how do you decide when one player has
too much of an advantage to allow the game to be admitted, and do you
remove the game from the list if the game is solved/known/believed to
give one player a winning advantage?
