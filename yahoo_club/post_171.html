Since it uses cards, Key Whiz doesn't look like what I'd call
an "abstract game." One feature that makes a game "abstract" is the
absence of randomizing elements and the presence of "perfect
information" about the current state of the game.

That said, it is possible to have an abstract game using
cards. "Mate," described in Sid Sackson's "A Gamut of Games," is an
example. But that's because in Mate both players, before they make
any decisions, are aware of who has which cards, and this is not true
of most card games. (Also, Mate is played in two rounds with the same
deal, the players swapping hands between rounds, so that neither
player has a disadvantage due to bad luck in the deal.)

This is not to say that Key West isn't a fine game, of course, but I
don't think it falls in the "abstract" classification.
