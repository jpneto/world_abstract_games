> Yes, you may see it so.

The reason for the random selection of games is so that it prevents
players from preparing for what they might face, so it provides a more
accurate test of their creative problem solving skills. If, say,
players play both sides in the game, and at the tournament know what
games they will be up against, and the criterion for who starts the
first game (before alternating) is based on player ranking or other
criterion, please inform me why this wouldn't count. Also, as far as
I can tell there is no such game as this moment called "Thompson
NonRandom Chess"

> > In such a tournament, would the fair thing be to have players play
> > both sides for each game?
>
> Not necessarily. If each game in the set is balanced (with chances
> for both players) then there is no need to play both sides.
> In Fischer Random Chess there is no setup known that gives one
> player an decisive advantage.

So, Fischer's Random Chess has been shown to be balanced enough for
both sides? Then it would be an abstract strategy game by the
criterion used here, correct, and an association for abstract strategy
games can manage it, right?
