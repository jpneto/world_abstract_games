The only trouble you point out in my definition is
the infinite
recursion. I am implicitly using the
convention that a fraction
is living if and only if you
can reach the conclusion that it is
living through
the definition. You are not supposed to think
that
"living" may have another meaning. If you really cannot
reach any
conclusion, the fraction isn't living.
Like a Turing Machine that
would accept its input
when and only when it is a living fraction
- if the
Turing Machine never halts, then the input
isn't
accepted and the fraction is said not to be
living.

Anyway, it looks like I wasn't clear so here is
another
definition that makes it explicit with geometric
concepts
(through the word "reaches"):

A stone or a
fraction of stone is *living* if and only if there
is
some path from it on the adjacency grid that reaches
an empty
intersection or a smaller opposing
fraction and that goes only
through intermediate
intersections occupied by friendly
fractions.
