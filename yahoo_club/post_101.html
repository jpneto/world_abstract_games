Idea for a new connection game

Author: markthomps

Date: 2001-08-08 23:58:55

Post ID: 101

I was thinking about Hex, and the well-known fact
that draws are impossible because on any
completely-filled board, either Black connects the Black edges or
White connects the White edges.

I asked myself,
"But what if you were to reverse the colors of the
edges? Clearly there still couldn't be a draw, which
tells us that in any completely-filled board, either
one color connects to ALL FOUR edges, or else BOTH
colors IN PARALLEL connect the same pair of
edges."

Now I'm wondering whether it would be possible to
construct a good game on this basis: Start with a Hex
board, probably a small one since we're all beginners --
maybe 4x4 or 6x6. But it will differ from a true Hex
board in that the edges are uncolored. Each player in
turn places both a White and a Black stone onto the
board. The object for the first player is to achieve two
parallel paths, one in each color, connecting opposite
edges of the board; the object for the second player is
to achieve a single group all in one color
connecting all four edges of the board.

I call this
game "Jade", because of my worry that its strategy
might be opaque. I've tried playing a couple small
games against myself, and it seemed playable -- I knew
what I was doing and why. If anyone else is interested
in trying it out (by yourself or with friends), I'd
be interested in hearing what you think of
it.
