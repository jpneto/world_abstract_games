No, Fischer's Random Chess is not an abstract game. As I already
answered you in another forum, a variant of an abstract game is not
necessarily also an abstract game. If a variant is produced by adding
random elements to an abstract game, then clearly the variant is not
an abstract game. There is no exception for Chess and there is no
exception for former Chess champion designers.

One would think that the "Random" in "Fischer's Random Chess" would be
a clue.
