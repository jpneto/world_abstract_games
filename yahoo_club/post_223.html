Online Twixt is now possible, BTW. If you have the software
package "Zillions of Games" you can download Twixt from their
website at http://www.zillionsofgames.com
Look for Twixt listed with the July 6 game releases.
My website has some screenshots and info at
http://www.geocities.com/twixtplayer/ZTwixt1.html

If you'd rather not pay for Zillions, there is a free Java
applet that actually PLAYS the game, albeit poorly, at
http://www.twixt.de
If you have the Java Runtime Environment (also free), you
can download their jar file and run it on your machine.
It's entirely in German. The 'pie' rule is NOT supported,
nor is link removal. It appears to have Internet
connection capability. If you'd like to play a German Java
Net Twixt game sometime, I'm willing to try it out with you!
