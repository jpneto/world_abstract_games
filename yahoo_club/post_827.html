> Stacking Games should be defined mathematically.
>
> Definition
> A "Stacking Game" is a game which can have >1 pieces at the same place that have different functions depending on their position, thus forming a "functional stack". Each functional stack is an independent one-dimensional unit (bottom - top).
>
> Examples
> Avalam Bitaka, Bashni, Byte, Dvonn, Emergo, Focus, Lasca, Pasta, Rondell, Stapeldammen, Tricolor.
