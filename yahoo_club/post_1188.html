> A second point: If a game is not a theoretical draw, then there has to
> be an advantage for one player or the other (obvious). There are
> several potential problems here: how do you decide when one player has
> too much of an advantage to allow the game to be admitted, and do you
> remove the game from the list if the game is solved/known/believed to
> give one player a winning advantage?

You run smack in the middle of the heap paradox problem. Exactly when
is something part of something, if there is no clearcut way of doing
it. The easy way out is to not think about it, call it quits and not
try to classify. This easy way out means you end up with not
bothering to start an association to promote abstract strategy games
and so on.
