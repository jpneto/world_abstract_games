Hello guys!
Have I talked about
www.ludoteka.com?
It is a web page where you can play some abstract
games in real time against other human players.
It
is a java applet so downloads are not required and
it's absolutely free.
One can
play:
Chess,chaturanga,spanish
checkers,connect-4,go-moku,hex,lines-of-action,quarto,surakarta,parchisi,reversi,tablut and wari
And some spanish card games like:
Pocha, escoba, sixes. And the very famous Basque game
called Mus.

It is available in Spanish, French,
English and Basque.

It's likely you won't find a
lot of people there since the page is
new.
