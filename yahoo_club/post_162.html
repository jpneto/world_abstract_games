This discussion has got me thinking of potential scoring methods.

What if the players, recieved a set point value for winning each game
but the recieve point values for each of their pieces captured.

The winning value could be about half the total value of the player's
pieces, thus forcing each player to sacrifice a fair number of men in
order to achieve a final winning score in the tournament.

There would also reach a point in the game where a losing player
would be better off if the winner continue to take pieces.
