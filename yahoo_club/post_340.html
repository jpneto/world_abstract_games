I've uploaded to the Files section the Zillions-of-Games
implementation of ZERTZ which I've been working on for several
months. I hope that the Group will test it for me.

Because Zillions is primarily a chess engine, it oftens doesn't play
alignment or accumulation game without far look-aheads on the turns.
I believe that I've worked the coding for this game so that this is
no longer necessary. Try running the game at Strength=Expert,
Variety=Large and Averge Time=1-3min.

This is still a test version, so I expect some constructive criticism.
