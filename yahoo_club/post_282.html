Catalyst: A New Abstract Strategy Game

Author: mschoessow

Date: 2002-08-28 20:20:36

Post ID: 282

I've recently posted the rule set for an unusual abstract strategy
game on the Board Game Design Forum at www2.communityzero.com/bgdf
(click on the Submitted Games section). The game is played on a 4x4
board, yet achieves good depth by combining mutable pieces with the
use of two special pieces called Catalysts that are used to activate
the powers of other pieces. The game has been reasonably well play-
tested so that I consider it "done", but I would still be interested
in hearing comments, pro or con, from anyone who's interest is piqued
enough to give it a try. The components required are a 4x4 board and
some Ice House pyramids (or 3 sizes in 2 colors of some other
stackable pieces).
