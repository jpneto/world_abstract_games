New piecepack game design contest announced

Author: Ron Hale-Evans

Date: 2006-09-21 18:45:43

Post ID: 1005

The rules for the seventh public game design contest for the piecepack
have been announced, and the theme is Good
Portsmanship. In the spirit of free and open source software and
culture, every entry must be a translation, or "port", of an existing
game to the piecepack. While this may seem like a mechanical exercise
at first glance, there is plenty of room for the game designer's
talents.

There was some debate on the piecepack mailing list about the original
rules of the contest, which some people thought were rather strict.
There have been several easements, and the rules are now at version 2.
They can be found here:

http://www.ludism.org/piecepack/pp7.html

Deadline for submissions is 3 December 2006, so you still have plenty
of time to enter.

Please forward this message to any other mailing lists, forums,
newsgroups, or friends you think are appropriate.
