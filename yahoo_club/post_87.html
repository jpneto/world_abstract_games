Abalone Comment
Submitted by Richard Irving Date: Apr 12, 2000
Abalone
- is it broken?

With the standard rules,
yes. The basic problem is, that with the standard set
up, it is far to easy to set up in a "ball" that
cannot be pushed back or outflanked to the edge so it
can be pushed off. You are only at risk in the game
if you try to attack. Once both players realize this
(or even one player who always plays to not lose and
has enough patience to never get greedy and actually
do something stupid like trying to win the game),
the game is dead.

Someone (I think it was
Mark Johnson) sent along an alternate set up used in a
tournament somewhere that splits each player's forces in
half. I never played it that way, but it would make it
harder to form a ball than in the initial set up, but it
might give the game too much of an advantage to the
first player--basically the game will be more of a race
to see which player can get into contact with the
opposing that he has surrounded at the start.

The
game is so actractive in a visual, auditory and
tactile sense. I just wish the game itself satisfied me
an intellectual sense.
