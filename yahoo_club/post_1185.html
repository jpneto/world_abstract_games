In perfect play Thompson Nonrandom Chess would be equivalent to game
number 480 in a list of games sorted by favorableness to Player 1,
game number 960 being most favorable. (That is, if Player 1 were
first to choose. Otherwise it would be game number 481.) Basically
it'd be a game of Nim followed by game number 480. Really not a bad
idea as this would serve to minimize move order advantage. But, then
again, perfect play would lead to a draw.
